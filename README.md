작성자: 신지용
날짜: 2025.11.28
설명: 이 프로젝트는 Unreal Engine 5를 사용하여 작성된 숫자야구 멀티플레이어 게임입니다.
주요 기능:
- 플레이어 간 채팅 기능
- 턴제 게임 플레이
- 게임 시작 지연 타이머
사용법:
1. 프로젝트를 Unreal Engine 5에서 엽니다.
2. 게임을 실행하고 멀티플레이어 모드로 접속합니다.
3. 채팅 입력창에 메시지를 입력하여 다른 플레이어와 소통합니다.
4. 자신의 턴이 되면 숫자를 추측하여 게임을 진행합니다.
요구 사항:
- Unreal Engine 5 이상
- C++20 호환 컴파일러
- 네트워크 연결
